# Wikipedia Tinder

A Next.js application that presents Wikipedia articles in a Tinder-like swipeable card interface.

## Features

- **Swipeable Cards**: Drag Wikipedia articles left or right with mouse or touch
- **Keyboard Controls**: Use arrow keys to navigate articles
- **Visual Feedback**: Green border for right swipes, red for left swipes with rotation effects
- **Article Navigation**: Predefined paths through related Wikipedia articles
- **Timer Feature**: Track time with starting and target article display
- **Modern Design**: Sleek, professional UI with smooth animations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

- **Swipe Right**: Navigate through the right swipe article path (ending at Israel)
- **Swipe Left**: Navigate through the left swipe article path
- **Arrow Keys**: Use ← and → keys to navigate
- **Timer Button**: Hover near the bottom-right corner to reveal the timer button
